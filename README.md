# ☀️ 太陽系シミュレーター

## 概要

`new/index.html` は、HTML5 と JavaScript（Canvas API）で作成された**太陽中心のインタラクティブな太陽系シミュレーター**です。

現在の実装では、太陽を中心にした惑星の公転を、Kepler の軌道要素に基づいて計算しています。

---

## 特徴 ✨

| 要素             | 説明 |
|------------------|------|
| 🌞 **太陽中心**      | 太陽を固定し、周囲を惑星が楕円軌道で回転します。 |
| 🪐 **惑星**        | 任意の惑星を追加・削除でき、名前・色・軌道要素を表示します。 |
| ⏩ **速度調整**      | シミュレーション速度を変更して、時間の流れを速くできます。 |
| 🔍 **ズーム・パン**   | マウスホイールでズーム、ドラッグで移動が可能です。 |
| 🧭 **軌道表示**      | 惑星の現在位置と軌道を描画し、軌道形状を視覚的に確認できます。 |
| 🖱️ **インタラクティブ** | クリックで惑星の情報を表示、ダブルクリックで新しい惑星を追加できます。 |

---

## ファイル構成

- `new/index.html` — メインの太陽系シミュレーター
- `new/README.md` — このドキュメント

> `new/index2.html` は削除済みです。

---

## 使い方

### 1. ブラウザで開く

Windows では `new/index.html` を右クリックしてブラウザで開くか、コマンドラインから:

```powershell
start .\new\index.html
```

### 2. 基本操作

- マウスホイール: ズームイン / ズームアウト
- ドラッグ: キャンバスをパン
- クリック: 惑星の情報を表示
- ダブルクリック: 新しい惑星を追加
- UI スライダー: シミュレーション速度を調整
